西雅图IT圈：seattleit

【今日作者】宇直

宇宙第一直男


根据外媒 Bloomberg 昨天报道，科技巨头微软已经同意收购在程序员间很受欢迎的代码库公司 GitHub Inc.。并且最快会在周一公布交易的细节。



根据熟悉该交易的内部人士透露，与其上市，GitHub 更希望将公司全额出售。GitHub之所以选择了微软是因为和微软的CEO  萨提亚·纳德拉聊的更开心。



该交易的具体细节目前还没有公布，GitHub 上一次的估值是在2015，当时的价格为20亿美元。


这次收购将为 GitHub 提供一条出路。去年8月以来，GitHub一直试图寻找新的CEO来代替当时的 CEO Chris Wanstrath，也是公司的创立者之一，其核心服务代码共享服务也一直没有盈利。



微软在这几年间一直断断续续和 GitHub 有过关于合作的讨论。也是直到最近，这种讨论才上升到收购。


▌ GitHub 是广大程序员们重要的编程工具之一。很多公司，包括微软，谷歌都使用 GitHub 来存储自家的代码以便实现代码合作。同时GitHub也是程序员之间一种社交的方式。



GitHub 目前存储着来自两千七百万程序员的超过八千万份代码库。微软曾经是反对开源代码的先锋，然而现在已经是 GitHub 上最大的贡献者之一。

这次收购也将帮助在越来越依赖开源项目的微软获取更多的开源资源。自从纳德拉成为CEO依赖，微软在开源项目上面的投资增长迅速。PowerShell, Visual Studio Code 以及 Microsoft Edge JavaScript engine 都已经被微软开源。作为微软从 Windows 向 Linux 系统开发的一个趋势，收购 GitHub 将给予微软更多和广大程序员交流合作的空间。
