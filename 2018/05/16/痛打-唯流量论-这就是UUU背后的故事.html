我一听傻眼了，UUU（U Network）？这不是硅谷密探的长期战略合作方，资源完全互享的 U Network 吗？


本是同根生，相煎何太急，我这是黑一下好呢？还是黑一下好呢？


想了一下，不知不觉，硅谷密探已经3年了。2015年5月4日，硅谷密探发布了第一篇文章。3年了，我们在硅谷为大家深度报道了不下500家科技创业公司，为啥这么要跟 U Network (UUU) 结成长期战略合作关系呢？


U Network 背后到底是一盘怎么样的大棋？为了维护世界的和平？毕竟 U Network 这个项目默默背后被硅谷投资机构 Draper Dragon、斯坦福丹华资本、PreAngel、D fund、Block VC、量子链等30多个中美顶级和(tou)平(zi)使(ji)者(gou)支持着...



探长说了，首先有这么一个原因...


世界正在悄悄地变化...


我们在内容产业默默耕耘的过程中，发现了几个问题：


信息过载已经成为人们的痛点，信息轰炸越来越多，而精品的内容越来越少。在免费互联网时代，我们享受了免费带来的便利，但是当所有人青睐免费的产品，内容的质量就会难以保障。因为作者需要生存，公司需要营利。流量变现或许能部分解决头部作者内容变现的问题，但是并不能解决大量腰部作者内容变现的问题。


当前有大量内容生产者花费了大量的时间和精力，生产了优质的内容，但是却没有获得合理的回报。站在天平另一端的读者，则面对着大量没有营养的内容，需要花费大量的时间去筛选，同样是受害者。


这个问题在倡导免费的互联网时代一直没有得到很好的解决。在当前互联网的基础设计上，我们并没有把这种支付的能力写入到底层协议的代码里面，所以你没办法直接在协议层支付购买你想要的商品或服务。


而区块链的出现，有史以来我们第一次把经济系统内嵌到了互联网里。去中心化的区块链和 token 代表的是一种更直接的经济系统，打破只有靠“流量变现”的诅咒。


这个世界正由免费互联网向价值互联网迁移。U Network（UUU）想做的，就是做价值内容第一公链，将内容资产化，让内容价值回归。U Network 打造的去中心化内容发布和估值平台，直接将内容生产者创造的价值回馈给内容生产者，不再被平台方剥削。


好吧，用人话说，就是让作者赚到钱！


打破流量思维的诅咒


仅仅是让作者赚到钱是不够的，我们还要让饱受“流量思维”的荼毒的内容创作者真正能够真正获得解放。


我们举个例子：打滴滴顺风车的空姐遇害案尚未平息，但不少人都听说了微信公众号 “二更食堂” 为了蹭热点、抢流量、吸引眼球，不惜写出一篇内容低俗、格调低下的文章，以至于被勒令停号整改的事情。


为什么会发生这样的情况？还不是对“十万加” 的狂热追求闹的？如果好内容能够给作者、甚至早期慧眼识珠的读者带来回报，那作者也就不用想方设法靠 “蹭热点” 吸引眼球了，而是能更专心地生产高质量的内容。


流量思维不仅荼毒了内容生产者，还荼毒了所有的读者！


U Network： 价值内容第一公链


U Network 致力于发现有价值的内容，而 Steemit 那种 “大鲸效应” 和机器刷赞在 U Network 上统统失效，充斥 Steemit 的低质量内容在 U Network上也不会被鼓励。所谓 “大鲸效应”，指的是社群里代币越多的“大鲸”，说的话就越有分量、因此也更容易左右别人的意志。一些没被大咖卡上的优质内容、优质内容产生者，就很有可能被遗落。


U Network 一个很有趣的机制就是，它的点赞是需要成本的。在以往的内容点赞机制中，点赞由于没有成本，在很大程度上带有随意性，如果这个是有成本的，那么读者的点赞可能更为慎重，而根据点赞的内容排序可能就更合理。



而点赞又是可以获利的，如果读者较早地投票早点赞，后面有很多人跟着他点，早期投票的读者是可以获利的。可以把原创内容比作一个创业项目，点赞即是一种投资，最早期就是天使投资、种子轮投资、A轮、B轮、C轮……就像风险投资助力于挖掘优秀创业者，而 U Network 也致力于挖掘优秀内容，并给读者合理的回报。


好吧，用人话说就是，写垃圾文章凑数，用机器人刷赞，在U Network上没有用的！


U Network要回归到内容产业的本质，挖掘和产生优质内容，让用户真正获得价值。


为什么是U Network？


探长说，硅谷密探是区块链领域真正的布道者，而硅谷密探将长期为 U Network 赋能。


这个我举双手同意，毕竟在去年的9月4号之后一周多，我们就头条发了一篇文章《区块链，将如何重新定义世界？》，是的，9月4号之后。



从早在2016年，我们就邀请以太坊创始人V神来布道以太坊（当时V神还没什么名气，是个满脸青春痘的羞涩小孩）；



到邀请币圈著名投资人老猫来分享区块链的实践；再到2017年7月，密探与火币集团COO朱嘉伟联合推出目前中文世界里最好的区块链入门课程《从0到1全面学透区块链》，已经被网易云课堂、喜马拉雅FM、一块听听等多个知识付费平台上架；


此外，我们还和UIUC计算机系博士董沫推出了《智能合约开发课》，默默培养了300多名区块链开发者。﻿


而 U Network 是一支真正有内容基因的国际化团队，对内容产业有深刻的理解，对内容产业的迭代和演进拥有独特的认知，了解内容消费者和内容生产者的需求。


U Network 背靠硅谷密探长期运营的区块链社区，有丰富的社区运营经验，拥有20万优质的区块链爱好者社区，以及300余人的区块链开发者社区。



U Network 在公链开发上拥有丰富经验的开发团队和顾问团队，而国内首屈一指的开发者社区，更是未来的杀手锏。


悄悄透露一下，未来 U Network 会和耶鲁大学等海内外高校产生合作科研合作，在公链开发和共识算法的研究上产生深度合作。
