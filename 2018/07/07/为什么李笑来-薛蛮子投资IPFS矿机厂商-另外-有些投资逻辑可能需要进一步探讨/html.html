【备注】

IPFS（InterPlanetary File System），星际文件系统是一个面向全球的、点对点的分布式版本文件系统，目标是为了补充（甚至是取代）目前统治互联网的超文本传输协议（HTTP），将所有具有相同文件系统的计算设备连接在一起。原理用基于内容的地址替代基于域名的地址，也就是用户寻找的不是某个地址而是储存在某个地方的内容，不需要验证发送者的身份，而只需要验证内容的哈希，通过这样可以让网页的速度更快、更安全、更健壮、更持久。 【来自巴比特】

当SDS遇见BlockChain 之二：区块链存储为什么势在必行？（SDS的新赛道 - 暗流涌动的区块链存储）

区块链创业和投资分析：早期阶段哪些项目可能性较大？


详细内容可以阅读上面两篇文章。这里，我汇总强调一下九存创始团队很早就得出的结论：


1、区块链一定能发展起来，但基础设施要先行


全球级别的资源共享，包含共享计算、共享存储、共享网络）。共享计算，现在全球2100个数字货币之下的公链，有许多公链实现的正是共享计算（只是我们通常仅关注数币而已）；共享网络，如NKN、迅雷玩客币等；当下，DApp所需的区块链基础设施，其中存储是瓶颈，因此区块链存储的需求和市场机会很大。


2、区块链存储项目将像雨后春笋般冒出来


在过去几个月内，我已经听到了数十个个DApp项目，都在其演讲中，把IPFS做为其至关重要的一个协议层。这是因为公链上存放的数据只有几个字节或者几KB的字节，不足以支撑DApp的繁衍。但是，具有长远战略眼光的区块链从业者、投资者，不要只把目光盯在一个IPFS（其实IPFS应用生态已经逐步完善了，但挖矿的激励体系，Filecoin还没上线。截止今天为止，加上我在文章《存储数币（如IPFS Filecoin）挖矿跟BTC/ETH挖矿有何不同？》里提到的，最近几个月就要公布的，以及昨天（2018-7-7）迅雷公布推出区块链文件系统TCFS（Thunder Chain File System），已经超过了20多个项目。备注：TCFS类似IPFS，也属于区块链存储的范畴内。【消息来自链闻】


3、区块链存储的发展分为两个方向


一是从上往下，先有公链，例如IPFS、StorJ，再往下走看看有没有发力点，例如研发、设计、生产并销售矿机。作为矿工需要注意的是，如何不被忽悠？

1）全球级别的项目，一般不会出台矿机标准

有越多的存储矿机为这个全球化的存储公链项目提供存储资源，项目方自然越高兴；

2）存储矿机厂商，如果提到所谓几个月回本，其实不靠谱

数币运行了较长时间，才有可能进入相对稳定期（注意还是相对的！），对应的法币值才有可能达到一个相对的平衡态势，此时计算收入产出，才有可能相对精确一些。数币发行初期时，很难计算，比如有些数币迅猛增长几百倍、几千倍，而有些数币几个震荡之后，几乎归零。IPFS Filecoin尚未上线，更不可能算出何时回本，目前在交易所上的是民间个人或团体推出的期货价格！


从上往下不是不可能，但我相信这里面但难度非常巨大。因为，真正的区块链项目一上来就是全球项目，就要和全球顶尖的团队所从事的项目进行直接竞争。另外，原来在存储软件上有着研发积累的团队，其实在IPFS等区块链存储项目的矿机领域内，并不能占据多少优势。原因是IPFS等项目，已经实现了绝大部分的软件功能，也就是说，当你做为IPFS矿机节点加入的时候，你原有的存储软件已经而且必须被屏蔽，否则效率不高，或者运维异常复杂（因为隔了1层，难以监控和诊断故障盘）。此时，硬件创新，尤其是像九存刀片存储支持并行挖矿，将逐渐成为构建企业级存储矿场的必备功能。


4、存储的硬件创新没有你想象的那么容易


存储的硬件创新其实门槛很高，很多年前，全球只有几大存储厂商巨头：EMC, HDS, NetApp, IBM, HP，为什么呢，因为硬件技术没有多年的积累是不可能的。到了2012～2014年，软件定义存储的兴起，开始有些百花齐放的感觉，但大多数仍属于软件的创新，大多数存储初创公司OEM使用DELL、HP等大型IT厂商的存储服务器，只有少数存储公司做了硬件创新，但多数只是在一、两个点上，例如高密度，例如低功耗。而九存有五个以上的亮点，而且申请十多个专利。九存是基于过往十多年的数据中心经验和企业级存储经验，极具勇气和魄力迈开了这一步，也是顺应时代的需求，例如云计算和大数据催生的历史数据在线化，需要刀片这种架构，能够有更高的并发读写的性能。


5、区块链项目要和实体经济结合起来


区块链存储解决了实际世界中的痛点，例如：存储空间不够用，成本较高；中心化云存储可能存在隐私泄漏的问题。再加上区块链存储还能天然的助力，数字资产的确权、分享即价值交换，这使得存储数币在交换这个使用价值之外，还有其他潜在的巨大的价值，而且挖矿所需的能耗较小，有望收到更多的民间个体、团队甚至政府的欢迎。详见：《工信部于佳宁：区块链要服务实体经济》；《存储数币（如IPFS Filecoin）挖矿跟BTC/ETH挖矿有何不同？》。
